Recursos

Uma central moderna para marcar amistosos.

Amistosos e solicitações

Publique data, horário e local; outros times podem solicitar e o responsável aceita ou recusa.

Quadras com mapa

Cadastre locais para selecionar nas partidas e evitar conflito de horário na mesma quadra.

Placares e estatísticas

Os dois times confirmam o resultado; divergências vão para revisão administrativa.

Ranking e reputação

Vitórias, aproveitamento, saldo de gols, fair play e confiabilidade ajudam na escolha do adversário.

Feed dos times

Publicações com fotos, curtidas, comentários e moderação para criar comunidade.

Admin completo

Painel para acompanhar usuários, times, quadras, denúncias, mídias, placares e selos.

FutMatch Landing page publicada via GitHub Pages. O aplicativo Flask completo precisa de hospedagem Python.
